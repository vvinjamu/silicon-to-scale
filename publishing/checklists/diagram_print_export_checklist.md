# Diagram Print Export Checklist

Use this checklist for every diagram before print publication.

## Required Checks

- [ ] Export SVG/HTML diagram to 300 DPI PNG
- [ ] Create vector PDF version if possible
- [ ] Verify diagram fits 7x10 trim size
- [ ] Verify text is readable in print
- [ ] Verify diagram still works in grayscale
- [ ] Add figure number
- [ ] Add caption
- [ ] Add alt text for web
- [ ] Place figure near first discussion in chapter
- [ ] Confirm no clipping in print PDF

## Output Folders

- assets/diagrams/png_300dpi/
- assets/diagrams/pdf_vector/
- assets/diagrams/webp/

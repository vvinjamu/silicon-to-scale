# Print-Ready KDP Checklist

## Goal

Prepare a print-ready interior and cover package for Amazon KDP or another print vendor.

## Recommended Settings

- Trim size: 7x10 preferred
- Alternate trim: 8.5x11 for diagram-heavy version
- Interior: premium color if budget allows
- Diagram resolution: 300 DPI minimum
- Body font: 10.5 to 11 pt
- Code font: 8 to 9 pt
- Captions: 8.5 to 9 pt
- Margins: mirror margins with gutter
- Bleed: only if diagrams/images extend to page edge

## Interior Checklist

- [ ] Final page size selected
- [ ] Margins and gutter configured
- [ ] No table overflow
- [ ] No code overflow
- [ ] Diagrams readable in print
- [ ] Fonts embedded
- [ ] Page numbers correct
- [ ] TOC page numbers correct
- [ ] No orphan headings
- [ ] No broken captions
- [ ] Export print PDF to final/print/

## Cover Checklist

- [ ] Final page count known
- [ ] Spine width calculated
- [ ] Full-wrap cover created
- [ ] Front, back, spine included
- [ ] 0.125 inch bleed included if required
- [ ] Barcode area left blank if using KDP ISBN
- [ ] Export cover PDF to final/print/

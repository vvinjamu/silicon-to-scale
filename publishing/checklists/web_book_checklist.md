# Web Book Checklist

## Goal

Turn the current GitHub Pages site from a PDF landing page into a readable online book.

## Required Tasks

- [ ] Create chapters/ folder for HTML chapters
- [ ] Create chapter page template
- [ ] Add sidebar navigation
- [ ] Add previous/next chapter links
- [ ] Add figure captions and alt text
- [ ] Add mobile responsive layout
- [ ] Add SEO title and description per chapter
- [ ] Add Open Graph metadata for social sharing
- [ ] Add diagram gallery page
- [ ] Add appendices as HTML pages
- [ ] Validate all links
- [ ] Test on Chrome, Edge, Safari/mobile
- [ ] Add search if possible

## Recommended Web Structure

chapters/
  ch00.html
  ch01.html
  ...
  ch18.html

appendices/
  appendix_a.html
  appendix_b.html
  appendix_c.html

assets/
  css/
  js/
  images/
